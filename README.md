# Simple Image Viewer
Personal Project to teach myself how to make Chrome Web Extension

### Shortcuts
* [R] - Rotate CW 90 degrees
* [Shift + R] - Rotate CCW 90 degrees
* [1] - Actual size
* [2] - Fill
* [3] - Fit
